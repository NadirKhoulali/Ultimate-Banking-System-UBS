Ultimate Banking System (UBS)

A complete banking economy for NeoForge 1.21.1.
Bring realistic finance gameplay to survival, roleplay, and server economies.

Highlights
  • Secure multi-account system (Checking, Saving, Money Market, Certificate)
  • ATM UI with PIN protection, account switching, and transaction history
  • Physical USD bill items ($1, $2, $5, $10, $20, $50, $100)
  • Player-owned banks + central-bank regulation and reserve management
  • Bank Owner PC with desktop-style management apps
  • Developer API + placeholders for scoreboard/HUD/shop integration

Core Gameplay Systems

  • Deposits, withdrawals, transfers, pay requests, and scheduled banking logic
  • Daily limits, anti-abuse guardrails, transaction trails, and account controls
  • Bank notes and cheques with issuer/recipient tracking
  • Bank Teller NPC flow for cheque processing

Bank Owner PC

  • Windows-inspired full-screen banking workstation
  • Bank dashboard sections for reserve, governance, staffing, lending, and compliance
  • Utility apps: Calculator, Notepad, Paint, File Explorer, System Monitor

Developer Support

  • Public API for account/bank snapshots, cash operations, and instrument issuance
  • Placeholder resolver for integrations and custom overlays
  • Full docs and integration tutorial in the project wiki
Status note: Bank Heist is currently marked Coming Soon and intentionally disabled in this release.
